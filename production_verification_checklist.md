# 員工出勤管理系統 - 生產環境上線後驗證清單 (Post-Deployment Verification)

本清單用於系統成功部署到 Render 生產環境後，進行功能面的黑箱與白箱冒煙測試（Smoke Tests）。

---

## 🟦 1. 生產網址與系統健康度檢查 (Domain & Health Check)

- [ ] **連線健康度監控**
  - [ ] 存取 `https://[您的生產域名]/api/health`。
  - [ ] 確認 HTTP 狀態碼為 `200`，且 JSON 欄位中 `"status": "UP"`、`"database": "connected"`。
- [ ] **SSL 安全傳輸確認**
  - [ ] 確認瀏覽器網址列顯示鎖頭標誌，憑證安全有效。

---

## 🟦 2. 首位管理員與登入後台測試 (Admin Access & Management)

- [ ] **初次登入驗證**
  - [ ] 前往 `https://[您的生產域名]/admin/login`。
  - [ ] 輸入預設初始管理員帳號及密碼。
  - [ ] 成功登入並跳轉至 `/admin` 首頁儀表板。
- [ ] **立即變更密碼安全強化**
  - [ ] 進入 **員工管理** 頁面或自行在資料庫中為 admin 變更一組非預設的密碼，確保生產安全。

---

## 🟦 3. 打卡點 GPS 與 QR Code 海報驗證 (Clock Panel & QR Poster)

- [ ] **QR Code 產生與列印海報**
  - [ ] 前往管理後台的 **打卡 QR Code** 頁面。
  - [ ] 確認打卡網址顯示符合當前網域的 `https://[您的生產域名]/clock?token=WORKPLACE_TOKEN`。
  - [ ] 下載打卡 QR Code，確認圖片解析度清晰。
  - [ ] 點選列印，確認 A4 排版樣式正常，無導覽列遮蔽。
- [ ] **公開打卡面板存取**
  - [ ] 使用手機掃描列印出的 QR Code，前往打卡網頁。
  - [ ] 確認網頁顯示「三峽八方雲集國際店」，且不需進行管理員登入即可開啟。
  - [ ] 測試 GPS 定位權限授權，確認能正常取得手機經緯度座標。

---

## 🟦 4. 員工出勤與薪資報表測試 (Functional Flow Check)

- [ ] **員工資料設定**
  - [ ] 在後台建立一位測試員工，設定其時薪（例如 `190`）。
- [ ] **打卡行為模擬**
  - [ ] 在店內範圍（或透過經緯度模擬）使用測試員工進行「上班打卡」與「下班打卡」。
  - [ ] 確認打卡成功，並能看到「上班成功」及「下班成功」的 Traditional Chinese 回饋資訊。
- [ ] **出勤記錄查核**
  - [ ] 登入管理後台的 **出勤紀錄**，確認出現該名測試員工今日的打卡時間、GPS 距離、IP 地址與 NORMAL 狀態。
- [ ] **薪資統計與報表匯出**
  - [ ] 前往 **薪資報表** 頁面。
  - [ ] 點選 **匯出全員薪資 Excel** 與 **匯出選定員工 Excel**。
  - [ ] 確認瀏覽器順利下載檔案，檔案名稱格式為 `三峽八方雲集國際店_薪資報表_YYYY_MM.xlsx`。
  - [ ] 用 Excel 開啟，驗證 `薪資總表`、`出勤明細`、`摘要` 等三個分頁工作表存在，且格式排列正確。
